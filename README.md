

# MiTool.exe
>### MiTool help即可列出命令列表
>####Just a tool for mi auth....

## 命令列表：
#### 1.help (帮助)
#### 2.sendloader {引导文件地址} {端口号}
#### 3.auth {内存类型} {方式} {端口号}
例：
```
MiTool help
```
```
MiTool sendloader I:\alioth_images_V13.0.9.0.SKHCNXM_20221102.0000.00_12.0_cn\images\prog_ufs_firehose_sm8250_ddr_5.elf 3
```
```
MiTool auth ufs 2 3
```
```
MiTool auth ufs 1 3
```
